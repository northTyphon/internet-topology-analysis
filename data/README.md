# Dataset

This project uses the CAIDA AS Relationships dataset (November 2007).

- Source: `https://snap.stanford.edu/data/as-Caida.html`
- Original: `https://www.caida.org/catalog/datasets/as-relationships/`
- License: CAIDA Acceptable Use Agreement

## Setup

1. Download `as-caida20071105.txt` from the SNAP link above
2. Place it in this folder renamed as `dataset.txt`
3. Run `python main.py` from the project root
